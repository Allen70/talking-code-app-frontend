Talking Code Denver

Site Description

Hi there! This is a place for the students and alumni of the "love bubble" to practice talking code with each other.


It's also a repository of resources to help us as we make our way through each mod and beyond.


And, it's got a directory of the places where we can find each other online so that we can stay in touch.
