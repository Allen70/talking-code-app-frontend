Talking Code Denver

Talking Code Denver is a place for the students and alumni of the "love bubble" to practice talking code with each other. It is also a clearing house, of sorts, of resources to help us as we make our way through the program. And, lastly, it includes a directory of our contact information so that we can stay in touch.
